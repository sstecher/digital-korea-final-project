**digital-korea-final-project**

Repository for the BA2: Digital Korea final paper at Leiden University.

**Research Question**

What recurring themes and emotional tones appear in Korean colonial-period poetry, and how do they differ across poets?

**Project Description**

This project analyzes recurring themes and emotional tendencies within Korean colonial-period poetry using the “kpoem” corpus. Topic modeling and sentiment analysis were applied to identify recurring thematic patterns, emotional tendencies, and similarities and differences across poets from the Japanese colonial period.

The analysis focuses on symbolic language, emotional ambiguity, and indirect emotional expression within colonial-period Korean poetry. Themes such as uncertainty, reflection, identity, longing, and nature imagery appear throughout the corpus, while sentiment analysis shows clustering around neutral to slightly negative emotional values.

**Repository Structure**

* analysis/ — analysis workflow and computational methods
* data/ — dataset information and data dictionary
* figures/ — figures included in the final paper

**Reproducing the Analysis**

The analysis was conducted using preprocessing, topic modeling, and sentiment analysis on the “kpoem” corpus. Figures included in the final paper were generated through this workflow and are included in the repository.

**Methods Used**

* Kiwi tokenization
* POS filtering
* Stopword removal
* LDA topic modeling
* Sentiment analysis
* Visualization through histograms and boxplots

**Corpus**

The project uses the “kpoem” corpus, which consists of 615 poems written in Hangul by Korean poets during the Japanese colonial period.
